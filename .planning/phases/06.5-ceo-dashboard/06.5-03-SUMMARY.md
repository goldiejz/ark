## md5 Invariant Proof

```
$ md5 -q ~/vaults/ark/observability/policy.db   # before
52db44655a2b432bb0aba4b75c536da6
$ ./target/release/ark-dashboard
ark-dashboard scaffold OK · policy.db reachable
$ md5 -q ~/vaults/ark/observability/policy.db   # after
52db44655a2b432bb0aba4b75c536da6   ← byte-identical
```

The binary opens the database with `SQLITE_OPEN_READ_ONLY` at the rusqlite C-API layer. Even a buggy SQL statement in 06.5-04 cannot mutate the file because the kernel-level fd is opened O_RDONLY by the SQLite C library when this flag is set. This is defense-in-depth on top of the bash `-readonly` discipline established in 06.5-01.

## Decisions Made

See frontmatter `key-decisions`. Highlights:

1. **ratatui + crossterm version-paired at 0.28** — Mismatched versions break event-stream re-export at the type level; the plan's specified pairing is the only safe combination.
2. **Bundled SQLite over system libsqlite** — One-time 25-30s build cost vs forever-broken macOS version drift. Worth it.
3. **No async runtime** — Synchronous local-file SQLite reads; async would add 50+ deps and zero throughput.
