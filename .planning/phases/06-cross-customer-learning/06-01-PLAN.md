  <action>
    Create `scripts/lib/lesson-similarity.sh` (executable, `chmod +x`).

    1. **Header**: shebang `#!/usr/bin/env bash`. Module comment naming the phase
       (Phase 6 Plan 06-01, REQ-AOS-32) and locked decision (D-SIMILARITY:
       Jaccard token overlap on title + rule body). State Bash-3 compat
       constraint up front. `set -uo pipefail` (NOT `set -e` — sourced lib must
       not break callers).

    2. **Stop-word list** (LOCKED — 50 most common English function words plus
       lesson-formatting noise):
       ```
       a an and are as at be but by do for from has have he her him his i if
       in is it its not of on or our she that the their them they this to was
       we were will with you your trigger mistake rule date lesson
       ```
       Store as a single `STOP_WORDS=" a an and are ... lesson "` string
       (space-padded both ends; allows `*" $tok "*` lookup — Bash 3 friendly).

    3. **lesson_tokenize "<text>"**:
       - Lowercase via `tr '[:upper:]' '[:lower:]'` (NOT `${var,,}` — Bash 4 only).
       - Strip non-alphanumeric via `tr -c 'a-z0-9' ' '`.
       - Squeeze spaces, split on whitespace via `tr -s ' ' '\n'`.
       - Drop empties and stop-words via awk: `awk -v sw="$STOP_WORDS" 'NF { if (index(sw, " " $0 " ") == 0) print }'`.
       - Drop tokens of length < 2 (single chars are noise).
       - Output one token per line. Do NOT `sort -u` here — caller decides.

    4. **lesson_extract_body "<lesson.md>"**:
       - If file does not exist or is empty, echo nothing and return 0 (the
         caller's Jaccard will compute 0 against any non-empty body).
       - Detect format A (strategix convention) by grepping for `^## Lesson:`.
         If matched, extract the FIRST `## Lesson:` block: from that line up
         to (but not including) the next `^## ` line or EOF. From within the
         block, emit:
         - The title line content (text after `## Lesson: `).
         - The Rule field content (the line(s) starting `**Rule:**` until the
           next `**Field:**` or blank line).
       - Detect format B (plain heading) by grepping for first `^# ` line.
         If matched, emit the first heading text + the first non-blank
         paragraph after it.
       - Use `awk` state machines for the block extraction — single pass, no
         temp files. Quote everything; lessons may contain backticks, dollars,
         quotes.

    5. **lesson_similarity "<a.md>" "<b.md>"**:
       - Pull body A and body B via `lesson_extract_body`.
       - Tokenise each (still as line-stream).
       - Build the **set** for each via `sort -u`.
       - Compute `|intersection|` via `comm -12 <(...) <(...)` piped to
         `wc -l`. Compute `|union|` via `cat ... | sort -u | wc -l`.
       - Compute integer percent via awk:
         `awk -v i="$inter" -v u="$union" 'BEGIN { if (u==0) print 0; else printf "%d\n", (i*100)/u }'`.
       - On any read error or empty union, echo `0` and return 0 (not non-zero —
         the caller treats 0 similarity as a normal result).

    6. **Self-test entry** (only when executed, not sourced — guarded by
       `if [[ "${BASH_SOURCE[0]}" == "${0}" ]]; then`):
       Use `mktemp -d` for fixtures (Phase-4 GitHub-incident lesson — no
       writes to real vault).
       Cases:
         - `assert_eq 100 "$(lesson_similarity $A $A)"  "identical file → 100"`
         - `assert_eq 0   "$(lesson_similarity $A $X)"  "disjoint vocab → 0"`
         - `assert_eq 100 "$(lesson_similarity $A $A_caps)"  "case-only delta → 100"`
         - `assert_eq 100 "$(lesson_similarity $A $A_stopwords)"  "stop-words only delta → 100"`
         - `assert_eq 50  "$(lesson_similarity $half_overlap_a $half_overlap_b)"  "1-of-2 token overlap → 50"`
         - `assert_eq 0   "$(lesson_similarity /nonexistent/a $A)"  "missing file → 0"`
         - Real-lesson sample: build an A from a `## Lesson:` block and B from
           the same block with 3 random words substituted; assert similarity
           between 60 and 99 (NOT 100, NOT 0 — sanity range).
         - Bash-3 compat scan (mirrors policy-learner.sh footer):
           `grep -c -E '(^|[[:space:]])(declare -A|mapfile|readarray)([[:space:]]|$)' "$0"`
           must return 0. **Self-referential test pattern trap (Phase 4
           lesson):** the grep must exclude comments and the test region
           itself. Restrict scan to lines BEFORE the `if [[ "${BASH_SOURCE[0]}"
           == "${0}" ]]; then` line (use the same `awk '/.../ { exit } ...'`
           prefix-scan as policy-learner.sh).

    7. **CLI usage**: when executed with no arg, print one-line usage:
       `Usage: $0 test | tokenize <text> | extract <lesson.md> | similarity <a> <b>`.
       Each subcommand calls the matching public function. Useful for ad-hoc
       debugging and for 06-05's verify suite.

    Avoid: ML libs (no python, no node), external HTTP, /dev/urandom (no need
    here), associative arrays, mapfile/readarray, `${var,,}`. The whole module
    is portable Bash 3 + coreutils + awk + sort/comm.
